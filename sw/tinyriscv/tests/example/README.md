本目录包含当前处理器使用的 C 语言示例程序和公共启动代码。

主要内容：

- `asm`：汇编相关示例。
- `cnn`：CNN 相关示例。
- `coremark`：CoreMark 基准测试。
- `cpu_test`：综合 CPU 功能测试示例。
- `dhyrstone`：Dhrystone 基准测试。
- `fft`：FFT 示例。
- `file_trans`：文件传输相关示例。
- `full_complian`：完整指令/功能覆盖示例。
- `full_complian_cnn_c`：结合 CNN 的完整示例。
- `FreeRTOS`：FreeRTOS 例程。
- `os`：简易 OS 示例。
- `vido_uart`：视频/串口相关示例。
- `include`：公共头文件。
- `lib`：公共库函数。

本目录根下的 `start.S`、`trap_entry.S`、`init.c`、`trap_handler.c`、`d.lds`、`common.mk` 等文件会被各个示例共享。
